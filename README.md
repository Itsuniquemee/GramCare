# GramCare

GramCare is a Node.js web application with static frontend pages and API-backed behavior for core workflow management.

## Tech Stack
- Node.js
- Express-style server setup
- HTML/CSS/JavaScript frontend

## Features
- Lightweight server with static page delivery
- Local JSON-backed data workflow
- Dashboard and login views in public folder
- Fast local development experience

## How to Run
1. Clone the repo
   git clone https://github.com/Itsuniquemee/GramCare.git
2. Enter project folder
   cd GramCare
3. Install dependencies
   npm install
4. Start server
   node server.js
